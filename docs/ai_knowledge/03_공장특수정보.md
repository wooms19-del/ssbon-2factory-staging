## 수율 측정 방식 (중요)

### 회사 표준 = 원물 대비 누적 수율 (stage-to-stage 아님)
- 전처리 수율 = 전처리 산출 / **원물 무게**
- 자숙 수율 = 자숙 산출 / **원물 무게** (전처리 산출 대비 아님)
- 파쇄 수율 = 파쇄 산출 / **원물 무게**

### 정상 범위
- **원육 수율 (최종)**: 50~58%
- **포장 수율**: 105~110% (소스 첨가)
- **포장 불량률**: 1% 이하

## 데이터 매칭 룰

### Thawing/날짜 매칭
- `thawing.date` = **종료일** (= 작업일, 박스 출고일)
- `thawing.start` = 전날 오후 시작 (16~17시간 전)
- `barcode.date` = **입고일/검수일** (thawing보다 1일 전인 게 일반적)

### 같은 (date, type) 공유 제품
- 같은 날짜 + 같은 부위 + 다른 제품 (예: 시그니처 일반 vs 마트용) → **같은 원물 공유**
- 일별실적 표에서 박스/원육은 rowspan 병합 (1회만 표시)

## 공정 흐름 (요약)

```
[입고] barcode (importCode 등록)
    ↓
[방혈/해동] thawing (16~17시간)
    ↓
[전처리] preprocess (비가식부 제거)
    ↓
[자숙] cooking (4시간 삶기)
    ↓
[파쇄] shredding (결대로 찢기)
    ↓
[포장] packing (파우치 충진)
    ↓
[외포장] outerpacking (박스 포장)
```
