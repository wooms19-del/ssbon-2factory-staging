## 캐시 / 배포 3-set 룰

**절대 분리 금지** — 항상 함께 처리:

1. **코드 push** (GitHub)
2. **index.html의 `?v=` 번호 증가** (해당 파일)
3. **Firestore `_config/version` PATCH** (auto-reload 트리거)

스크립트 형태:
```bash
NEW_V=$(date +%s)
sed -i "s|FILE.js?v=[0-9]*|FILE.js?v=$NEW_V|" index.html
git add -A && git commit && git push
curl PATCH _config/version with value=$NEW_V
```
